# 🎙️ Jarvis Voice Assistant

Jarvis Voice Assistant is a Python-based virtual assistant that performs everyday tasks using voice commands. It can recognize speech, respond using text-to-speech, search the web, open applications, play music, tell the current time, provide weather updates, and automate common desktop activities.

## ✨ Features

- 🎤 Voice command recognition
- 🗣️ Text-to-Speech responses
- 🌐 Search the web
- 🔎 Google search integration
- ▶️ Play YouTube videos
- 🎵 Play music
- 📂 Open desktop applications
- ⏰ Tell current date and time
- 🌦️ Weather information
- 📧 Send emails (configurable)
- 🤖 Interactive conversational responses
- ⚡ Easy to extend with new commands

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Riya-Madhumatke/Jarvis-voice-assistance.git
```

### 2. Navigate to the project

```bash
cd Jarvis-voice-assistance
```

### 3. Create a virtual environment (Optional)

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux / macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

```bash
python main.py
```

Once started, Jarvis will begin listening for voice commands.
